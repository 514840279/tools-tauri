# tools-tauri
基于tauri构建第一个桌面项目

## I 项目情况

### 1. 采用 naive ui 作为主题样式

``` ts
npm i -D vfonts
npm i -D naive-ui
npm i -s vue-router@next 
npm i -s axios
npm i -s vue-axios
npm i sass  node-sass sass-loader  --save-dev 
npm i -s echarts
npm i -s moment  
npm i -s pinia
yarn add pinia-plugin-persist
```

2. 启动 
```
npx vite --port=4000
yarn run dev
yarn tauri dev
```
